# Mobile Subscription App

A modern mobile subscription app built with **Expo** and **React Native**. The app allows users to explore subscription plans, manage their subscriptions, and interact with a clean mobile-first user interface.

This project was inspired by this YouTube tutorial:
https://www.youtube.com/watch?v=4nVoLX2taFg

## Features

* Modern mobile app interface
* Expo-based React Native setup
* File-based routing with Expo Router
* Subscription plan screens
* Reusable components
* Mobile-first layout
* Easy project setup and development workflow

## Tech Stack

* **React Native**
* **Expo**
* **Expo Router**
* **TypeScript**
* **JavaScript**
* **npm**

## Getting Started

Follow the steps below to run the project locally.

### 1. Clone the Repository

```bash
git clone <your-repository-url>
```

### 2. Navigate into the Project Folder

```bash
cd <project-folder-name>
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npx expo start
```

After running the command, Expo will provide options to open the app on:

* Android Emulator
* iOS Simulator
* Expo Go
* Development Build

## Running on a Device

To preview the app on a physical device:

1. Install **Expo Go** from the App Store or Google Play Store.
2. Run:

```bash
npx expo start
```

3. Scan the QR code shown in the terminal or browser.

## Resetting the Project

To reset the starter project structure, run:

```bash
npm run reset-project
```

This will move the starter code into the `app-example` directory and create a fresh `app` directory.
